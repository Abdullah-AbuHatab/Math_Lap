# BaseTenBlocks - دليل الاستخدام الشامل

## 📚 المقدمة

تطبيق BaseTenBlocks هو أداة تعليمية تفاعلية لتعليم مفهوم النظام العشري (Base-10/Place Value) باستخدام بلوكات ملموسة بصرياً.

---

## 🎮 الأنماط المتاحة

### 1. وضع Basic Mode
للممارسة الحرة والتجريب بدون قيود.

**كيفية الاستخدام:**
- اضغط على الكتل في الشريط الجانبي لإضافتها
- اسحب الكتل لتحريكها
- استخدم زر السياق (يمين الماوس) للإجراءات الإضافية

### 2. وضع Place Value Mode ⭐
نمط متقدم مع جدول ذكي ينظم الكتل حسب المنازل:

```
┌─────────────────────────────────────────────┐
│   Thousands │ Hundreds │ Tens │ Ones        │
├─────────────────────────────────────────────┤
│             │          │      │             │
│   (Cubes)   │ (Flats)  |(Rods)|(Units)     │
│             │          │      │             │
└─────────────────────────────────────────────┘
        ↓
    Total: 2547
```

**المميزات:**
- ✅ الجدول يترتب تلقائياً حسب قيمة الكتلة
- ✅ تفكك ذكي للكتل الكبيرة
- ✅ منع الأخطاء (لا يمكن وضع كتلة كبيرة في مكان صغير)
- ✅ حساب تلقائي للقيمة الإجمالية

## 🧮 أنواع الكتل

- **القيمة:** 1
- **اللون:** أصفر (#eab308)

### 🟠 Flat (المئات)

- **اللون:** برتقالي (#f97316)
- **الحجم:** 140×140 بكسل
- **الشكل:** شبكة 10×10 (100 مربع)
- **الاستخدام:** تمثيل 100 وحدة

### 🟢 Cube (الآلاف)
- **القيمة:** 1000
- **اللون:** أخضر (#22c55e)
- **الحجم:** 160×160 بكسل
- **الشكل:** 3D مع شبكة أمامية 10×10
- **الاستخدام:** تمثيل 1000 وحدة

## 🎯 الإجراءات الأساسية

### ➕ إضافة كتلة
**الطريقة 1: النقر البسيط**
```
1. اضغط على الكتلة في الشريط الجانبي
2. ستضاف مباشرة للمكان المناسب
```

**الطريقة 2: السحب والإفلات**
```
1. اسحب الكتلة من الشريط الجانبي
2. أفلتها حيث تريد
3. ستترتب تلقائياً في المنطقة الصحيحة
```

### 🎯 اختيار كتلة
```
• نقر مفرد = اختيار كتلة واحدة
• Shift + نقر = إضافة/إزالة من الاختيار
• Ctrl/Cmd + نقر = اختيار متعدد
• اسحب + افلت = اختيار مربع (Marquee)
```

### 🔄 نقل كتلة
```
1. انقر واسحب الكتلة
2. في Place Value Mode: ستترتب تلقائياً في المكان الجديد
3. في Basic Mode: ستلتصق بأقرب كتلة (snap to grid)
```

### ⚙️ قائمة الخيارات (右クリック)
```
يمين الماوس على كتلة مختارة:

• Break (تفكيك)
  - Cube → 10 Flats
  - Flat → 10 Rods
  - Rod → 10 Units

• Regroup (إعادة تجميع)
  - 10 Units → 1 Rod
  - 10 Rods → 1 Flat
  - 10 Flats → 1 Cube
  (اختر 10 كتل نوع واحد أولاً)

• Copy (نسخ)
  - ينسخ الكتلة المختارة

• Lock/Unlock (قفل/فتح)
  - منع/السماح بنقل الكتلة

• Delete (حذف)
  - إزالة الكتلة
```

## 📊 تحكم الجدول (Place Value Mode)

### تحريك الجدول
```
اسحب على الجدول ذاته (ليس على كتلة):
→ سيتحرك الجدول كاملاً
```

### تغيير حجم الجدول
```
اسحب "مقبض التغيير الحجم" (الزوايا والأطراف):
→ يتسع أفقياً وعمودياً
```

### عرض/إخفاء التسميات
```
قائمة الجدول (أيقونة المزيد) → Labels
→ إظهار/إخفاء عناوين المنازل
```

### نسخ الجدول
```
قائمة الجدول → Copy
→ ينسخ الجدول كاملاً مع محتوياته
```

### حذف الجدول
```
قائمة الجدول → Delete
→ حذف الجدول (لا يؤثر على الكتل المفقوسة)
```

## 🔍 التصغير والتكبير

### التحكم بالـ Zoom
```
• زر (+) في الأداة العلوية: تكبير
• زر (-) في الأداة العلوية: تصغير
• النسبة: 50% → 100% → 200%
```

### المزايا المتقدمة
```
• جميع العناصر (Mats، Blocks، الجدول) تتسع معاً
• الاتجاهات الطبيعية تُحفظ
• السحب والنقر يعملان بشكل طبيعي
```

## 🎓 حالات الاستخدام التعليمية

### 📖 درس 1: فهم النظام العشري
1. اضف 5 units
2. اضف 3 rods (كل واحد = 10)
3. اضف 2 flats (كل واحد = 100)
4. اضف 1 cube (= 1000)

**النتيجة:** 1235 ✅

### 🔄 درس 2: التفكك والإعادة
1. اسحب cube إلى عمود Hundreds
2. انظر: يتفكك تلقائياً إلى 10 flats
3. اسحب flat إلى عمود Tens
4. انظر: يتفكك إلى 10 rods

### ➕ درس 3: الجمع
1. اضف 245 (2 flats + 4 rods + 5 units)
2. اضف 187 (1 flat + 8 rods + 7 units)
3. لاحظ التغييرات الذكية (12 rods → 1 flat + 2 rods)

## 🐛 استكشاف الأخطاء

| المشكلة | السبب | الحل |
|--------|-------|------|
| الكتل تظهر خارج الجدول | اسحبها بعيداً جداً | أعد جر الكتل للداخل |
| لا تر الجدول | قد تكون في وضع Basic | انقل إلى Place Value Mode |
| الكتل لا تترتب | قد تكون مقفولة | استخدم قائمة الخيارات لفتح القفل |
